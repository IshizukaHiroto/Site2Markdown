Site2Markdown — Chrome Extension

プライバシーポリシー

最終更新日: 2026年3月21日

はじめに

Site2Markdown（以下「本拡張機能」）は、WebページをMarkdown形式に変換するChrome拡張機能です。

Site2Markdown は個人情報を一切収集・保存・外部送信しません。すべての処理はブラウザ内でローカルに完結します。

取り扱う情報

保存する情報（設定値のみ）

ユーザーが設定画面で変更した以下の設定値を chrome.storage.sync に保存します。この情報はあなた自身のChromeアカウントを通じてデバイス間で同期され、外部サーバーには送信されません。

  • 除去オプション（広告・ヘッダー・フッター・ナビゲーション・サイドバー）
  • 画像・リンクの処理方法
  • 見出しレベルのシフト設定
  • フロントマターの項目設定（タイトル・URL・日付・タグ等）
  • ファイル名フォーマット設定
  • その他の変換オプション

収集しない情報

  • 閲覧履歴・訪問したURLの記録
  • 変換したページの内容
  • 個人を特定できる情報（氏名・メールアドレス・IPアドレス等）
  • クリップボードの内容
  • ダウンロードしたファイルの内容

ページコンテンツへのアクセス

本拡張機能は、ユーザーがアイコンをクリックした瞬間にのみ、現在表示しているページの内容を読み取ります。読み取ったコンテンツはブラウザ内でMarkdownに変換され、ポップアップに表示するためだけに使用されます。

コンテンツはいかなる外部サーバーにも送信されません。バックグラウンドでの監視・記録も行いません。

使用するパーミッションと目的

パーミッション使用目的
activeTabアイコンをクリックしたときのみ、現在のページのコンテンツにアクセスします。他のタブやバックグラウンドへのアクセスは行いません。
storage変換設定を保存・復元します。
clipboardWrite「コピー」ボタン押下時にMarkdownをクリップボードへ書き込みます。
downloads「保存」ボタン押下時に .md ファイルをダウンロードします。
scripting変換ライブラリ（Readability・Turndown）をページに動的注入します。

同梱するサードパーティライブラリ

以下のオープンソースライブラリがブラウザ内でローカルに動作します。外部へのデータ送信は行いません。

  • Mozilla Readability — Apache 2.0 ライセンス — 記事本文の自動抽出
  • Turndown — MIT ライセンス — HTML→Markdown 変換
  • Turndown Plugin GFM — MIT ライセンス — GitHub Flavored Markdown 対応

第三者へのデータ提供

ユーザーのデータを第三者に販売・提供・共有することはありません。

Pro版ご利用時（将来予定）: 決済処理にExtensionPay（Stripe）を利用する予定です。決済情報はStripeに直接送信され、開発者はカード番号等の機密情報を取得しません。詳細は Stripeのプライバシーポリシー をご参照ください。

データの保持と削除

  • 本拡張機能をアンインストールすると、保存された設定データはすべて削除されます。
  • 設定画面から「デフォルトに戻す」で個別にリセットできます。

お子様のプライバシー

本拡張機能は13歳未満のお子様を対象としておらず、意図的にお子様の個人情報を収集することはありません。

ポリシーの変更

本プライバシーポリシーは必要に応じて更新する場合があります。重要な変更がある場合は、拡張機能のアップデートノートに記載します。

Site2Markdown — Chrome Extension

Privacy Policy

Last updated: March 21, 2026

Overview

Site2Markdown is a Chrome extension that converts web pages to Markdown format.

Site2Markdown does not collect, store, or transmit any personal data to external servers. All processing happens locally within your browser.

Information We Handle

Information We Store (Settings Only)

The extension saves the following user-configured settings to chrome.storage.sync, which syncs across your devices via your Chrome account. This data is never sent to any external server.

  • Cleanup options (remove ads, headers, footers, navigation, sidebars)
  • Image and link handling preferences
  • Heading level shift setting
  • Frontmatter field configuration (title, URL, date, tags, etc.)
  • Filename format settings
  • Other conversion options

Information We Never Collect

  • Browsing history or visited URLs
  • Content of pages you convert
  • Personal identifiable information (name, email, IP address, etc.)
  • Clipboard contents
  • Downloaded file contents

How We Access Page Content

The extension reads the current page content only when you click the extension icon. The content is converted to Markdown entirely within your browser and displayed in the popup.

Page content is never sent to any external server. The extension performs no background monitoring or logging.

Permissions and Their Purpose

PermissionPurpose
activeTabAccesses the current page content only when you click the icon. No access to other tabs or background pages.
storageSaves and restores your conversion settings.
clipboardWriteWrites Markdown to the clipboard when you click "Copy".
downloadsDownloads the .md file when you click "Save".
scriptingDynamically injects conversion libraries (Readability, Turndown) into the page.

Third-Party Libraries

The following open-source libraries run locally in your browser. They do not transmit data externally.

  • Mozilla Readability — Apache 2.0 License — Article content extraction
  • Turndown — MIT License — HTML to Markdown conversion
  • Turndown Plugin GFM — MIT License — GitHub Flavored Markdown support

Data Sharing

We do not sell, share, or disclose any user data to third parties.

Regarding future Pro plan (planned): Payment processing will use ExtensionPay (powered by Stripe). Payment information is sent directly to Stripe. We never receive or store your credit card details. See Stripe's Privacy Policy for details.

Data Retention and Deletion

  • Uninstalling the extension removes all stored settings data.
  • You can reset individual settings from the options page at any time.

Children's Privacy

This extension is not directed to children under 13 and does not knowingly collect personal information from children under 13.

Changes to This Policy

We may update this Privacy Policy from time to time. Material changes will be noted in the extension's update release notes.

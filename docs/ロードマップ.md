# 機能ロードマップ

作成: 2026-03-20

---

## 基本方針

- **v1.x**: 無料版として品質を磨く。ユーザー・レビューを集める
- **v2.0**: フリーミアム化。Pro 機能を解放し収益化を開始
- **v2.x 以降**: ユーザーの声をもとに機能を拡張

---

## v1.0（現在 / 公開版）

### 実装済み
- [x] Mozilla Readability による本文抽出
- [x] Turndown + GFM プラグインによる Markdown 変換
- [x] フロントマター（title / url / date / description / author / tags / custom）
- [x] 除去オプション（広告・ヘッダー・フッター・ナビ・サイドバー）

### 変更内容

## バージョン管理ルール

- セマンティックバージョニング: `MAJOR.MINOR.PATCH`
  - PATCH（1.0.x）: バグ修正・軽微な改善
  - MINOR（1.x.0）: 新機能追加（後方互換あり）
  - MAJOR（x.0.0）: フリーミアム化など大きなビジネスモデル変更
- GitHub の Releases にタグとリリースノートを記録
- Chrome Web Store はアップデートごとに再審査（通常 24〜72 時間）

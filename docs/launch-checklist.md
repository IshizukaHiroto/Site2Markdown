# 公開前チェックリスト

作成: 2026-03-20

---

## フェーズ1: コード・動作確認

### 基本動作
- [ ] 一般的なブログ記事（英語・日本語）で変換が正常に動作する
- [ ] Readability が失敗するページで body フォールバックが動作する
- [ ] `chrome://` など制限ページで適切なエラーメッセージが表示される
- [ ] コピーボタンが正常に動作する（クリップボードに入る）
- [ ] 保存ボタンで `.md` ファイルがダウンロードされる
- [ ] 設定を変更・保存し、ポップアップを開き直しても設定が引き継がれる
- [ ] フロントマターが正しい YAML 形式で出力される（コロンを含むタイトルでも崩れない）

### 設定項目の動作確認
- [ ] 除去オプション（広告・ヘッダー・フッター・ナビ・サイドバー）が機能する
- [ ] `imageHandling: altOnly / remove` が正しく動作する
- [ ] `linkHandling: textOnly` が正しく動作する
- [ ] `headingShift: 1 / 2` が正しく動作する
- [ ] `collapseBlankLines` が 3行以上の空行を 2行に圧縮する
- [ ] `selectionOnly` で選択範囲のみ変換される
- [ ] `autoCopy` でポップアップ起動と同時にコピーされる
- [ ] フロントマターの各フィールド（description / author / tags / custom）が出力される
- [ ] ファイル名フォーマット（日付形式・区切り文字・最大文字数）が反映される

### ブラウザ・環境
- [ ] Chrome 最新版で動作する
- [ ] Chrome 109（最小サポートバージョン）で動作する
- [ ] 拡張機能をアンインストール → 再インストール後も正常に動作する
- [ ] manifest.json をローカルで読み込んでエラーが出ない

## フェーズ2: ストア提出素材

### 必須素材
- [ ] アイコン: 16px / 48px / 128px PNG ✅（既存）
- [ ] スクリーンショット 4〜6枚（1280×800 または 640×400）
  - [ ] ① 変換後のポップアップ全体（英語ページ）
  - [ ] ② フロントマター付き出力
  - [ ] ③ 設定画面（本文整理タブ）
  - [ ] ④ 設定画面（メタ情報タブ）
  - [ ] ⑤ 変換前後の比較（任意）
  - [ ] ⑥ Obsidian への貼り付け例（任意）
- [ ] プロモーション画像（1400×560）

### テキスト素材
- [ ] 短い説明文（131文字・英語）→ `store-listing.md` 参照
- [ ] 詳細説明文（英語・パーミッション使用理由を含む）→ `store-listing.md` 参照
- [ ] `manifest.json` の `description` を短い説明文と統一する
